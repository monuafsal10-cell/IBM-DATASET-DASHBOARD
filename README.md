📊 IBM Sales Report Dashboard
📌 Overview

The IBM Sales Report Dashboard is an interactive data visualization project designed to analyze employee metrics, departmental performance, and key HR indicators. This dashboard provides meaningful insights into workforce distribution, income trends, attrition, and job satisfaction across different departments.

It is built to support data-driven decision-making by presenting complex data in a simple, visually engaging format.
